AI Playlist Curation

Deep Learning Audio Analysis Android Studio Deezer API

Développement d'un système de recommandation musicale basé sur la classification automatique des genres et des moods via l'analyse de spectrogrammes.

Code Source

L'Application Mobile

[ Vidéo/Image Démo Android App ]

Workflow Utilisateur

  1. Enregistrement audio ou choix d'un titre.
  2. Extraction des features (Tempo, Énergie).
  3. Classification Genre/Mood en temps réel.
  4. Génération de playlist via l'API Deezer.

Intelligence Artificielle

Datasets

  • GTZAN : 1000 clips pour la classification de base des genres.
  • MTG-Jamendo : 55,000 titres pour une classification multi-labels fine (instruments, moods).

Modèles

VGG16 (CNN) Random Forest SVM Ensemble Learning

Traitement du Signal :

L'innovation majeure réside dans l'utilisation de Librosa pour transformer l'audio en Spectrogrammes de Mel. Ces images sont ensuite traitées par un réseau de neurones convolutif (VGG16) pour capturer les motifs temporels et fréquentiels.
